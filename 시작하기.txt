HWPX 제안서 생성 시스템 - 시작하기
====================================================================

📌 3단계만 기억하세요!

1️⃣ Markdown 파일 작성
   - _examples\예시제안서.md 참고
   - AI 관련 키워드는 **볼드**로 작성 (자동 녹색)
   - 수치는 **볼드**로 작성 (자동 빨강)

2️⃣ 배치 파일 실행
   - 파일을 배치 파일에 드래그 앤 드롭
   - 또는: ALL_IN_ONE.bat 제안서.md

3️⃣ HWPX 파일 확인
   - 제안서.hwpx 파일이 생성됨
   - 한글 오피스로 열기

====================================================================

🎨 자동 색상 적용 규칙

녹색 (AI/기술):
  - AI, 인공지능, 자동화
  - 머신러닝, 딥러닝
  - LLM, GPT, 생성형
  - 리터러시, 역량
  - 시스템, 솔루션

빨강 (수치/성과):
  - 80%, 50% 절감
  - 40시간, 30분

예시:
  **AI 기반 시스템** → 녹색
  **80% 절감** → 빨강

  - sample-from-hangul.hwpx  템플릿 (절대 삭제 금지!)
  - proposal-styles.json     스타일 설정

⚠️ 문제 해결

Q: 한글이 안 보여요
A: sample-from-hangul.hwpx 파일 확인

Q: 색상이 안 나와요
A: Markdown에서 **볼드** 사용 확인
   키워드가 규칙에 맞는지 확인

Q: python/node 오류
A: Python, Node.js 설치 필요
   pip install python-hwpx lxml
