# HWPX 제안서 생성 시스템

AI 기반 제안서 작성 후 한글 오피스(HWPX) 파일로 자동 변환하는 시스템입니다.

## 🚀 빠른 시작

```cmd
ALL_IN_ONE.bat 제안서.md
```

위 명령 하나로 Markdown → JSON → HWPX 변환이 완료됩니다!

## ✨ 주요 기능

- ✅ **자동 색상 적용**: AI 용어는 녹색, 수치는 빨간색
- ✅ **표 자동 생성**: Markdown 표 → HWPX 표
- ✅ **한글 완벽 지원**: 한글 오피스에서 정상 표시
- ✅ **웹 미리보기 동일**: 색상/표 형태 일치
