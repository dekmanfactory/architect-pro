# HWPX 제안서 스타일 규정

## 📍 위치 안내

### 1. **스타일 설정 파일**
- 파일 경로: `proposal-styles.json`
- 역할: 모든 글자/문단 속성을 정의

### 2. **데이터 파일**
- 파일 경로: `proposal-example.json` 또는 `proposal-full.json`
- 역할: 실제 제안서 내용

---

## 🎨 글자 및 문단 속성

### Level 1 (□ 항목)
```json
{
  "symbol": "□",
  "font": "HY헤드라인M",
  "size": 16,
  "paragraphSpaceBefore": 15,
  "align": "justify",
  "leftMargin": 0
}
```
- **폰트**: HY헤드라인M
- **크기**: 16포인트
- **문단 위 여백**: 15pt
- **정렬**: 양쪽 정렬
- **왼쪽 여백**: 0pt
- **기호**: □

---

## 📝 수정 방법

### 1. 스타일만 변경하고 싶을 때
→ `proposal-styles.json` 파일 수정

예시:
```json
{
  "styles": {
    "level1": {
      "font": "맑은 고딕",     ← 폰트 변경
      "size": 18,              ← 크기 변경
      ...
    }
  }
}
```

### 2. 내용만 변경하고 싶을 때
→ `proposal-example.json` 또는 `proposal-full.json` 파일 수정

### 3. 변환 실행
```bash
node json-to-documents.mjs proposal-full.json
```
