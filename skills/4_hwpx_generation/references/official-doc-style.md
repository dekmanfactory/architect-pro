# 공문서(기안문) 양식 가이드

> 「행정업무의 운영 및 혁신에 관한 규정」(약칭: 행정업무규정) 및 동 시행규칙, 행정안전부 행정업무운영 편람을 기반으로 정리한 공공기관 공문서 작성 규격이다.

## 1. 용지 및 여백

### 1-1. 용지 규격
- **용지**: A4 (가로 210mm × 세로 297mm) — 규정 제7조제6항
- **방향**: 세로(Portrait)가 기본

### 1-2. 여백 (서식 설계 기준 — 시행규칙 별표 4)

| 항목 | 공식 서식 기준 | HWPUNIT |
|------|--------------|---------|
| 위쪽(상단) | 20mm | 5669 |
| 아래쪽(하단) | 10mm | 2835 |
| 왼쪽 | 20mm | 5669 |
| 오른쪽 | 20mm | 5669 |

## 2. 글꼴 및 글자 크기

### 기안문(공문서) 기본 서식

| 항목 | 글꼴 | 크기 | 장평 | 자간 | 비고 |
|------|-----|------|------|------|------|
| 본문 | 맑은 고딕 | 11.5pt | 100% | 0 | 2022년 이후 기본 |
| 기관명 | HY견고딕 | — | — | — | 공문 상단 |
| 줄간격 | — | — | — | — | 103% (맑은 고딕 기준) |

> **참고**: 2022년 이전에는 굴림체(12pt, 줄간격 123%)가 기본이었으나, 맑은 고딕으로 변경됨. 일부 교육청은 아직 굴림체를 사용하는 경우가 있다.

## 3. 항목 기호 체계

공문서의 내용을 둘 이상의 항목으로 구분할 때, 상위 항목부터 하위 항목까지 다음 순서로 표시한다 (시행규칙 제2조제1항):

```
1단계:  1.  2.  3.  4.  …
2단계:  가.  나.  다.  라.  …
3단계:  1)  2)  3)  4)  …
4단계:  가)  나)  다)  라)  …
5단계:  (1)  (2)  (3)  (4)  …
6단계:  (가)  (나)  (다)  (라)  …
7단계:  ①  ②  ③  ④  …
8단계:  ㉮  ㉯  ㉰  ㉱  …
```

### 표시 위치 및 띄우기 규칙
- **첫째 항목(1.)**: 띄어쓰기 없이 왼쪽 기본선에서 바로 시작
- **둘째 항목(가.)부터**: 상위 항목 위치에서 오른쪽으로 **2타씩** 들여 시작
- 항목기호와 내용 사이: **1타** 띄움 (예: `1. 내용`, `가. 내용`)
- 항목이 두 줄 이상이면 둘째 줄부터는 **항목 내용의 첫 글자에 맞추어 정렬**
- **항목이 하나만 있으면** 항목기호를 부여하지 않는다

```
수신  ○○○학교(○○○장)
(경유)
제목  ○○○○○

1. ○○○○○○○○○○○
  가. ○○○○○○○○○○○
    1) ○○○○○○○○○○○
      가) ○○○○○○○○○○○
        (1) ○○○○○○○○○○○
          (가) ○○○○○○○○○○○
2. ○○○○○○○○○○○○○○○○○○○○○○○○○○○○○○
   ○○○○○
```

> **"2타"**: 한글 1자 = 영문·숫자 2자에 해당

## 4. 날짜·시간·금액 표기법

### 4-1. 날짜 (규정 제7조제5항)
- 숫자로 표기하되, 연·월·일의 글자는 **생략하고 그 자리에 온점(.)** 을 찍어 표시
- 월·일 표기 시 **앞에 '0'을 표기하지 않음**
- 요일은 괄호 안에 표기

```
✕ 2026년 02월 13일 금요일
✓ 2026. 2. 13.(금)
```

### 4-2. 시간
- **24시각제**에 따라 숫자로 표기
- 시·분의 글자는 생략하고 그 사이에 **쌍점(:)** 을 찍어 구분

```
✕ 오후 3시 20분
✓ 15:20
```

### 4-3. 금액
- 아라비아 숫자로 쓰되, 필요 시 숫자 다음에 괄호하고 한글로 기재

```
✓ 금113,560원(금일십일만삼천오백육십원)
✕ 345천원
✓ 345,000원
```

## 5. "끝" 표시 및 붙임

### 5-1. "끝" 표시
- 본문의 마지막 글자에서 **2타 띄우고 "끝"** 표시
- 본문이 오른쪽 한계선에서 끝났을 때: 다음 줄 왼쪽 기본선에서 2타 띄우고 "끝"
- 본문이 표로 끝나는 경우: 표 아래 왼쪽 기본선에서 2타 띄우고 "끝"
- 기재사항이 서식 중간에서 끝나는 경우: 마지막 다음 줄에 **"이하 빈칸"**

### 5-2. 붙임 표시
- 본문이 끝난 줄 다음에 '붙임' 표시 후 첨부물의 명칭과 수량 기재
- 첨부물이 **하나면 숫자 '1' 표시하지 않음**
- '붙임' 다음에 쌍점(:)을 **붙이지 않음**

```
……(본문)…… 주시기 바랍니다.

붙임  2026학년도 운영 계획 1부.  끝.

──── (첨부물 2건 이상) ────

……(본문)…… 주시기 바랍니다.

붙임  1. ○○○계획서 1부.
     2. ○○○서류 1부.  끝.
```

## 6. 관련 근거 표기

```
1. 관련: 00과-1234(2026. 2. 13.)호

──── (관련 2건 이상) ────

1. 관련
  가. 교육부-1234(2026. 2. 1.)
  나. 경기도교육청-5678(2026. 1. 10.), 「2026년 000 운영 계획」
```

## 7. 문장 부호 사용

| 부호 | 용법 | 예시 |
|------|------|------|
| 가운뎃점(·) | 대등/밀접한 열거 | 융·복합, 시·군·구 |
| 쌍점(:) | 앞 붙여, 뒤 1타 띄움 | 원장: 김갑동 |
| 물결표(∼) | 앞뒤 붙여 씀 | 4. 23.∼6. 15. |
| 홑낫표(「 」) | 법령 이름 인용 | 「초중등교육법」 |
| 겹낫표(『 』) | 낫표 안의 낫표 | |

## 8. 공문서 작성 원칙 요약

- **정확성**: 정확한 용어, 올바른 문법, 사실 왜곡 방지
- **용이성**: 상대방 입장에서 이해하기 쉬운 용어
- **성실성**: 성의 있고 진실하게, 적절한 경어 사용
- **경제성**: 1건 1매 원칙, 표준 기안문 서식 사용
- **한글 작성**: 「국어기본법」에 따라 한글로 작성, 필요 시 괄호 안에 한자·외국어 병기

## 코드 예시: 공문(기안문) 생성

```python
from hwpx.document import HwpxDocument

doc = HwpxDocument.new()

doc.add_paragraph("1. 관련: 교육정책과-1234(2026. 2. 1.)")
doc.add_paragraph("2. K-에듀파인시스템을 활용한 학교업무 개선 및 효율화 방안 마련을 위하여 아래와 같이 담당자 협의를 안내하오니 대상자가 참석할 수 있도록 협조하여 주시기 바랍니다.")
doc.add_paragraph("  가. 일시: 2026. 3. 5.(목) 15:00∼17:00")
doc.add_paragraph("  나. 장소: 경기도교육청 소회의실8(남부청사 4층)")
doc.add_paragraph("  다. 대상: K-에듀파인시스템 00분과 위원 및 업무 담당자 20명")
doc.add_paragraph("  라. 내용: K-에듀파인시스템을 활용한 학교업무 개선 및 효율화 정책 방향 모색")
doc.add_paragraph("  마. 협조사항")
doc.add_paragraph("    1) 원활한 회의 진행을 위해 14:50까지 참석자 등록 완료")
doc.add_paragraph("    2) 청사 내 주차 공간이 협소하므로 대중교통 이용 권장")
doc.add_paragraph("")
doc.add_paragraph("붙임  2026학년도 K-에듀파인시스템 00분과 위원 명단 1부.  끝.")

doc.save("official_document.hwpx")
fix_hwpx_namespaces("official_document.hwpx")
```
